HTML Comment

HTML comments are not displayed in the browser, but they can help document your HTML source code. Comments are also great for debugging HTML, because you can comment out HTML lines of code, one at a time, to search for errors.

Codes/Tags

HTML Comment

You can add comments to your HTML source by using the following syntax:

<!-- Write your comments here -->

Notice that there is an exclamation point (!) in the start tag, but not in the end tag.

Add Comment

With comments you can place notifications and reminders in your HTML code:

<!-- Write your comments here -->

<p>this is paragraph.</p>

<!-- Remember to add more information here -->

Example

<!DOCTYPE html>
<html>
<body>

<!-- Write your comments here -->

<p>this is paragraph.</p>

<!-- Remember to add more information here -->

</body>
</html>

Result

this is paragraph.
